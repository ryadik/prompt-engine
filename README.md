# Prompt Engine

*Генератор системных промптов для LLM на основе структурированного описания задачи.*

## Описание

Этот проект предоставляет инструментарий для автоматического создания **высококачественных системных промптов** для языковых моделей (LLM). Основан на идее, что эффективный промпт можно сгенерировать, если четко и структурировано описать задачу.

## Особенности

*   **Высокая детализация:** Сгенерированные промпты следуют структуре, аналогичной референсным (как в `default_prompt.txt`), обеспечивая конкретику и ясность.
*   **Обязательные принципы качества:** В каждый сгенерированный промпт вшита фиксированная "ДНК" из 9 принципов, включая `No abstraction, only specifics`, `Don't be lazy`, `Always be clear and concise`, `Ask questions if something is not understood`, и другие.
*   **Структурированный ввод:** Использует набор вопросов для получения четкого и полного описания задачи от пользователя, минимизируя неопределенность.
*   **Универсальность:** Может генерировать промпты для различных задач (код, текст, анализ, синтез и др.), сохраняя высокий стандарт качества.

## Использование

1.  Запустите чат с вашей любимой LLM (например, Qwen, Gemini).
2.  Вставьте содержимое файла `prompt_gen.md` как **системное сообщение**.
3.  Ответьте на вопросы из файла `prompt_questions.md`, описав вашу задачу.
4.  LLM сгенерирует **новый, готовый к использованию системный промпт**, оптимизированный под вашу задачу.

## Файлы проекта

*   `prompt_gen.md`: Генерирующий промпт. Его нужно использовать как системный промпт для LLM.
*   `prompt_questions.md`: Набор вопросов для структурированного описания задачи.
*   `refs/`: Директория с референсными промптами, использовавшимися как пример структуры и подхода (не для прямого использования в генерации).
